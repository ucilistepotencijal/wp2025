# 🐾 Šapotel - Sustav za upravljanje hotelom za kućne ljubimce

**Šapotel** je web aplikacija razvijena kao završni rad, namijenjena vlasnicima hotela za pse i njihovim klijentima. Sustav omogućuje jednostavnu rezervaciju smještaja, upravljanje profilima ljubimaca te pruža adminu potpunu kontrolu nad poslovanjem.

---

## ✨ Ključne funkcionalnosti

### 👤 Korisnički dio (Klijenti)
* **Registracija i Prijava:** Siguran sustav autentifikacije.
* **Moji Ljubimci:** Dodavanje, uređivanje i pregled profila vlastitih pasa.
* **Rezervacije:** Jednostavno kreiranje termina boravka uz odabir vrste usluge.

### 💼 Admin Panel (Upravljanje)
* **Nadzorna ploča (Dashboard):** Vizualni pregled ukupne zarade, broja rezervacija i trenutne popunjenosti hotela.
* **Danas u hotelu:** Brzi uvid u pse koji se trenutno nalaze na smještaju.
* **Upravljanje uslugama:** Dodavanje i promjena cijena hotelskih usluga (npr. dnevni boravak, noćenje, šišanje).
* **Administracija korisnika:** Pregled svih registriranih korisnika sustava.

---

## 🛠️ Tehnološki stog (Tech Stack)

* **Backend:** ASP.NET Core 8.0 (MVC)
* **Frontend:** HTML5, CSS3, Bootstrap 5 (za responzivan dizajn)
* **Baza podataka:** Entity Framework Core (SQL Server)
* **Autentifikacija:** ASP.NET Core Identity
